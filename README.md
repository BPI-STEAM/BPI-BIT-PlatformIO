+ ### Text Editor
    > We need to prepare a text editor in advance (VSCode or Atom)

     [Install Platform IO](/docs/BPI-BIT_PIO.md)

+ ### Add Espressif 32 Development Kit
     [Espressif 32 for Atom/VSCode](/docs/ESP32-PIO-Development-Kit.md)

## Add Espressif 32 Development Kit:

- Open PlatformIO(PIO) Home Page:
    1. `Platforms`>`Embedded`>`ESP32`
    2. Click `Espressif 32`
    3. Click `Install`
    4. Waiting...

![Step 1](/docs/pio-15.png)

![Step 2](/docs/pio-16.png)
